# Env-Manager
Quick Deployment to install and configure packages with ansible.
## Dependencies

```bash
ansible-galaxy collection install \
    community.general

ansible-galaxy install \
    gantsign.oh-my-zsh \
    stephdewit.nvm \
    rvm.ruby
```
